# Contact.me

Keep track of your contacts, all in one place, anytime, anywhere!

## Technologies

- Ruby on Rails
- React.js
- Materialize CSS

## Local Development

- Install npm dependencies bundle install
- To run the app: rails s
- To run the tests run: rspec
